# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-20

### Added
- Initial release
- GLM-5.1 API Python SDK
- CLI tools for quick operations
- API Proxy server with caching and rate limiting
- Batch processing capabilities
- Multi-language documentation (Chinese, English)
- Comprehensive unit tests
- Rate limiting with token bucket algorithm
- Disk-based response caching
- Configuration management system

### Features
- `glm-toolkit`: Main CLI tool for API operations
- `glm-proxy`: API proxy server
- Chat completion with streaming support
- Batch processing with multi-threading
- Request caching for improved performance
- Configurable rate limiting
- Progress tracking for batch operations

- README in Simplified Chinese (简体中文)
- README in Traditional Chinese (繁體中文)
- README in English
- Contributing guidelines
- Installation and usage guides
